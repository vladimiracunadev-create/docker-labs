# 🤝 Support

Guía breve para pedir ayuda en `docker-labs`.

## Antes de reportar

Revisa primero:

- [README](README.md)
- [User Manual](docs/USER_MANUAL.md)
- [Troubleshooting](docs/TROUBLESHOOTING.md)
- [Dashboard Setup](docs/DASHBOARD_SETUP.md)

## Cómo pedir ayuda

- abre un issue si hay un bug reproducible
- abre una discussion si quieres proponer una mejora o resolver una duda

## Qué incluir en un reporte útil

- lab o componente afectado
- sistema operativo
- versión de Docker y Compose
- comando ejecutado
- resultado esperado
- resultado real
- logs o captura del error

## Alcance del soporte

Este repositorio está orientado a:

- aprendizaje práctico
- prototipado local
- construcción progresiva de una plataforma modular

No promete soporte de producción ni SLA.
