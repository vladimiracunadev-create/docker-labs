# Support

## Como pedir ayuda

Si encuentras un problema en este repositorio, usa uno de estos caminos:

- abre un issue si detectas un bug reproducible
- abre una discussion si quieres proponer una mejora o una duda de uso
- revisa primero la documentacion tecnica y operativa del repo

## Que incluir en un reporte util

- laboratorio afectado
- sistema operativo
- version de Docker y Docker Compose
- comando ejecutado
- resultado esperado
- resultado real
- logs o captura del error

## Antes de reportar

Revisa especialmente:

- [README.md](C:/docker-labs/docker-labs/README.md)
- [docs/USER_MANUAL.md](C:/docker-labs/docker-labs/docs/USER_MANUAL.md)
- [docs/TROUBLESHOOTING.md](C:/docker-labs/docker-labs/docs/TROUBLESHOOTING.md)
- [docs/DASHBOARD_SETUP.md](C:/docker-labs/docker-labs/docs/DASHBOARD_SETUP.md)

## Alcance del soporte

Este repositorio esta orientado a:

- aprendizaje practico
- prototipado local
- construccion progresiva de una plataforma modular

No promete soporte de produccion ni SLA.
